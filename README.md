# Python-Game-Text-Based-OOP-Trial
# 🎮 THE LAST TOWER

A Python text-based survival adventure game with combat, loot, bosses, and a final boss at Day 25.

---

## 📌 GAME FEATURES

- 👤 Username input system  
- 🎭 Character selection (Joker, Mysty, Lloyd)  
- 🌲 Explore system  
- ⚔ Turn-based combat  
- 🏃 Run or Attack option  
- 💥 Damage system  
- 🎁 Loot system  
- 👾 Random enemies  
- ⚠ Boss every 5 days  
- 🔥 FINAL BOSS at Day 25  
- 🏆 Win / Lose endings  

---

## 🧩 HOW TO RUN

Make sure you have Python installed.

Run the game using:

```bash
python main.py
